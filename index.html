* RATE SIMULATION ONLY.
This calculator provides an indicative estimate based on standard local charges. Official quotation and booking confirmation can be issued only in writing by Erixmar. Additional charges, exceptions, unexpected events, non-covered ZIP codes, special conditions, and any service not included in the standard tariff may affect the final amount. If the pickup ZIP code is outside the covered areas, please ask for information: a manual written quotation is required.

Included standard logic: Handling, THC on GW, Offloading/Reloading on CW with minimum and maximum, APT Transfer on CW, optional Customs, Customs Visit, Insurance, FBL/FHL, EUR1, XRY, and pickup based on ZIP prefix. Unknown or uncovered ZIP ranges trigger a manual quote notice.

Quick usage note for agents

  1. Enter pickup ZIP code, GW, CW, and pieces.
  2. Select only the services needed for the shipment.
  3. Click Calculate simulation.
  4. If the page shows Manual quote required, Erixmar must issue the quotation in writing.
